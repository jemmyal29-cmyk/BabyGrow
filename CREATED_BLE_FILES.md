## 🔑 Critical Success Factors

### Don't Forget These!
1. **Baud Rate = 115200** (most common mistake)
2. **USB data cable** (not power-only)
3. **Board = ESP32 Dev Module**
4. **COM port selected**
5. **Upload Speed = 115200**

### Check These First
1. Device Manager shows COM port?
2. Arduino IDE recognizes board?
3. Serial Monitor shows output?
4. BLE device appears in phone scan?
